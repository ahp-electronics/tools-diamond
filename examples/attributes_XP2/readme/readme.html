Instructions to port to any Lattice FPGA device

Supported Devices: LatticeEC/ECP/XP/ECP2/ECP2M/XP2

 

Instructions to port example to EC, ECP and XP devices:

  1. Open the ispLEVER Project Navigator by selecting “Start – Programs - Lattice Semiconductor – ispLEVER Project Navigator”.
  2. In the “Sources in Project:” window double click on the device.
  3. In the “Device Selector” window select any LatticeEC,ECP-DSP,XP device, then click OK, Then in the “Confirm Change” window click Yes.
  4. In the “Sources in Project:” window double click on the “top.v” Verilog file.
  5. In the Editor window just below the header comment field uncomment the “`define XP” line and comment out the “`define XP2” lines as shown:

// `define XP2

`define XP

          Then save and close the editor.

  1. In the “Sources in Project:” window double click on the “clk_pll_all.v” Verilog file.
  2. I the Editor window just below the header comment field uncomment the “`define XP” line and comment out the “`define XP2” lines as shown:

// `define XP2

`define XP

          Then save and close the editor.
