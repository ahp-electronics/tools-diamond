Converting a Schematic Design to Another Device

You can first target this design to the new device. As some symbols are only valid for some specific devices, you may find that some symbols cannot be used for the new device. Replace these symbols with new symbols that support the new device, according to the list of Schematic Symbols. See List of Schematic Symbols.
