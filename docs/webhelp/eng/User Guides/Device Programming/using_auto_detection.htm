Detecting a Cable

You can use the Diamond Programmer Detect Cable feature to determine which cable and port you are using.

To detect a cable:
In the Cable Settings Tab, click Detect Cable. The Diamond Programmer detects all available cables connected to your PC, and lists the cable type, port settings, and descriptions in the Output console.
To detect or upgrade an FTDI cable:
1. To plug in FTDI cable, see Plugging the Cable into the PC.
2. In the Cable Settings Tab, click Detect Cable. The Diamond Programmer detects the HW-USBN-2B (FTDI) cable connected to your PC, FTUSB-0 as the port settings, and descriptions in the Output console.
3. Click on Firmware Tab. A new window displays fields for MachXO2 firmware and POWR607 firmware.
4. Select a JEDEC (*.jed) file to update device/devices.
Note
If firmware versions of two JEDEC files don't match, the software will display a warning message asking if you want to continue.
5. Click on Update Firmware to reprogram devices on cable to specified file.
