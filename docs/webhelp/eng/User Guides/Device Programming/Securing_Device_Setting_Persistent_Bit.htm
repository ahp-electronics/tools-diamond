Securing the Device and Setting the Persistent Bit

The Security and Persistent Fields Editor allows you to read the settings of the security and the persistent bit of the bitstream file and change the setting to ON or OFF.

To read the settings of the security and persistent fields:
1. Choose Tools > Security and Persistent Fields Editor.
2. Click the Browse button and select the bitstream data file.
3. Click Read to read the settings for Security and Persistent Bit.
The editor reads the settings and displays them in the Status cells.
4. To change a setting from OFF to ON or vice versa, click the status cell.
5. Do either of the following to save the changes:
To save the changes to the current data file, click Write (or click the arrow beside Write and then choose Overwrite). In the pop-up dialog box, click Yes to confirm that you want to overwrite the data file.
To save the changed data file to a different location or with a different file name, click the arrow beside Write and then choose Save As. In the Save As Data File dialog box, specify file name and location, and click Save.
6. A message box tells you that Write to file was successful. Click OK.
7. Click Close to exit the Security and Persistent Fields Editor.
