Unlocking Secure SPI Flash Support

If a custom SPI flash device or a SPI flash device that supports secure golden patterns sectors was secured as described in Setting up Dual Boot Golden Sector Protection Support, and you wish to reprogram the SPI flash, you must first unlock the device before reprogramming.

To unlock a secure SPI flash:
1. Create a new Diamond Programmer project as described in Creating a New Diamond Programmer Project or open an existing Diamond Programmer project as described in Opening an Existing Diamond Programmer Project.
2. Right-click on the row corresponding to the device you would like to edit, and choose Edit > Device Properties.
3. In the Operation dropdown list, chose SPI Flash Unlock Device.
4. Ensure that the Secure SPI Flash Golden Pattern Sectors box is unchecked.
5. Click OK.
6. In Diamond Programmer, choose Design > Program, or click on the toolbar.

The SPI flash is now unlocked and can be reprogrammed.
