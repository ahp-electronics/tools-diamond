Programming Platform Manager 2, MachXO2, MachXO3D, or MachXO3L/LF Devices Using I2C with External ASC

Diamond Programmer supports I2C programming of Platform Manager 2, MachXO2, MachXO3D, or MachXO3L/LF devices with external ASC. Diamond Programmer requires an FTDI 2232H chip installed on the customer board to bridge the Lattice HW-USBN-2B (FTDI) USB programming cable to the Platform Manager 2, MachXO2, MachXO3D, or MachXO3L/LF and ASC I2C port.

To program a Platform Manager 2, MachXO2, MachXO3D, or MachXO3L/LF Device Using I2C with External ASC:
1. Right-click on the row corresponding to the device you would like to edit, and choose Edit > Device Properties.
2. In the Access Mode dropdown list, choose I2C Interface Programming.
3. In the Operation dropdown list, choose desired operation.
4. Enter the I2C slave address of the Platform Manager 2, MachXO2, MachXO3D, or MachXO3L/LF device.
5. In the External ASC Options box, click + to add an external ASC / LPTM21L device.
6. Select the data file, operation, and I2C slave address.
7. Repeat steps 5 and 6 for each external ASC / LPTM21L device.
8. Click OK.
9. Download the design file.
