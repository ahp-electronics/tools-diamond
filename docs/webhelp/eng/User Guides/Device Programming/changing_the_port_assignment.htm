Programming the FPGAChanging the Port Assignment

When you first launch the Programmer software with the cables properly connected, it connects to the first available port that it detects. You can change the connecting port, as well as other cable options, using the Cable and Port Setting dialog box.

To change the port setup:
In the Cable Settings Tab, choose the options that you want. Changes are immediately applied.
If the cable is not connected or cannot be detected (if the board power in not on, for example), Programmer software displays an error message.
