Device Properties Dialog Box

The following options are available in the Device Properties dialog box:

Access Mode

Selects the mode for programming the device.

Operation

Lists the available operation modes for the device. Select one from the drop-down list.

Programming File

Selects the data file for programming or verify.

Reinitialize Part on Program Error

Reinitializes the part to its blank, or erased, state when a programming error occurs.
