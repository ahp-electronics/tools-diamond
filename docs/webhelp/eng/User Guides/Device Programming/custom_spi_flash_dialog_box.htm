Custom SPI Flash Dialog Box

The following options are available in the Custom SPI Flash dialog box:

Device Family

Dropdown list of custom device families.

Device Description

The name that was assigned to the device.

Device Name

This can be any alpha-numeric string that describes the device name (for example: SPI-M25P32).

Package

This can be any alpha-numeric string that describes the package (for example: 16-pin SOIC).

Device Vendor

The vendor of the custom SPI Flash device.

Device ID

This can be any alpha-numeric string that describes the device ID (for example: 0x15).

Byte Per Sector

Allows you to choose desired bytes per sector value from dropdown.

Protection Options ON

If the SPI Flash device supports sector protection, check this box to enable protection options. This will allow you to select Secure SPI Flash Golden Pattern Sectors in the Device Properties Dialog Box.
