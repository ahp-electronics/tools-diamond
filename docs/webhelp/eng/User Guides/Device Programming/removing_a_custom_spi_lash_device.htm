Removing a Custom SPI Flash Device

f you have already created a custom SPI Flash device, and you wish to remove it from your the Custom SPI Flash Dialog Box.

To edit the properties of a Custom SPI Flash device:
1. Right-click on the row corresponding to the device you would like to edit, and choose Edit > Custom Device.
2. In the Edit Custom Device Dialog Box, select SPI Serial Flash Custom, select the SPI Flash device you want to remove, and click Remove and OK.
