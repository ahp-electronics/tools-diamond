Control Register0 Editor Dialog Box

The following options are available in the Control Register0 Editor dialog box:

Device

Lists the projectâ€™s device. If a device is not displayed, use the browse button to locate the correct data file. Once a device is displayed, click Read to get the correct Max Bits/Digits information.

Read

Reads the data file values and displays it in the display field.

Save

Saves the data file.

Save As

saves the data file with another file name.

Close

Closes this dialog box.
