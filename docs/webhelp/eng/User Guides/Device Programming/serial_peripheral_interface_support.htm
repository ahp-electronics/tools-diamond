Serial Peripheral Interface (SPI) Flash Support

Diamond Programmer, combined with a Lattice cable fly-wire, supports the programming of Serial Peripheral Interface (SPI) flash devices. All Lattice FPGA devices supported by the Diamond software support SPI Flash.

Each Lattice FPGA device contains Lattice's Reveal IP. Diamond Programmer uses the JTAG interface to program the SPI flash device. Because of their bitstream compression capability, these FPGAs allow the use of smaller-capacity SPI memory devices.
