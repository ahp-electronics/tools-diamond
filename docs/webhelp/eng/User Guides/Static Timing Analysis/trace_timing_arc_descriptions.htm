TRACE Timing Arc Descriptions

The following table provides definitions of timing arcs for setup time, hold time, delays, and minimum pulse width that you encounter within the TRACE report (.tw1/.twr) file. Timing arcs refer to intra-cell path delaysâ€”the time it takes for a signal to flow from a given logic cell's input transition to a resulting output rise (or fall) output transition. TRACE timing analysis reports timing arcs as a part of constraint calculation.
