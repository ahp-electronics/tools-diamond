Floorplan View and Physical View Partition Support

Both Floorplan View and Physical View can display partition information to support the Incremental Design Flow. Partitions can be edited in Floorplan View, but not in Partition View.

Floorplan View loads partition information that has not yet been loaded into memory. Data is only loaded after the Translate process. See Displaying Partitions in Floorplan View.
Physical View can display partition information to support the Diamond Incremental Design Flow. Data is only loaded after the Translate process. See Displaying Partitions in Physical View.
