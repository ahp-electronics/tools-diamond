Using SDC Constraints in Place and Route

When you chose the “Use LPF Created from SDC in Project” strategy setting, constraints set in Lattice Synthesis Engine (LSE) LDC Editor or Synplify Pro SCOPE Constraints Editor will be applied during the Diamond software Place & Route process.

To use SDC constraints in Place & Route
1. Specify “Use LPF Created from SDC in Project” in your project’s strategy. Refer to Specifying Strategy Options.
Note
Refer to the Strategy Reference Guide for more information about LSE and Synplify Pro versions of the “Use LPF Created from SDC in Project” strategy.
2. Run the Synthesize the Design process. Refer to Running Processes.
3. Set the new LPF file as the active file. Refer to Managing Constraint Files.
4. Run the Place & Route Design process. Refer to Running Processes.
