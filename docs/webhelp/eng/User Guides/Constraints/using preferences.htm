Using Preferences

Preferences are constraints that guide the design toward performance goals. They can be created and edited using Diamondâ€™s six preference-editing views, or they can be created and edited manually in the logical preference file (.lpf) using a text editor. Preferences, unlike HDL attributes, are written to the .lpf file.

This section describes the major types of preferences and the role they play in the design flow. For complete descriptions of all preferences, see the Constraints Reference Guide.
