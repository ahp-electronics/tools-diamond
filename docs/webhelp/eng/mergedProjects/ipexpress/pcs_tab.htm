PCS

Description:

PCS (Physical Coding Sub-layer) supports advanced system level standards including RapidIO, Sonet, Gigabit Ethernet, 10 Gb Ethernet, Fibre Channel, and PCI Express.

Devices Supported:

ECP5UM, LatticeECP2M, LatticeECP3, LatticeSC, LatticeSCM

Version:

8.2

Revision History:

8.1: Added Control Setup tab to control file generation.

8.0: Added Frequency attribute for LatticeECP2M. Added Frequency attribute, updated port name, and changes to SRIO for LatticeECP3.

7.0: Updated to latest LatticeECP2 data sheet. Fixed refclk2fpga, refclk multiplier (8B SerDes mode) for LatticeECP3. Fixed ORCAstra daisy chain issue for LatticeSC/M.
