Distributed_ROM

Description:

Implements a Read Only Memory using LUT logic.

Devices Supported:

ECP5U, ECP5UM, LatticeEC, LatticeECP, LatticeECP2, LatticeECP2M, LatticeECP3, LatticeSC, LatticeSCM, LatticeXP, LatticeXP2, LIFMD, LIFMDF, MachXO, MachXO2, MachXO3D, MachXO3L, Platform Manager, Platform Manager 2

Version:

2.9

Revision History:

2.9: Added LIFMDF support.

2.8: Added LIFMD support. Added MachXO3D support.

2.7: Added ECP5 support.

2.6: Added MachXO3L support.

2.5: Added support for comments and empty lines in memory initialization files.

2.4: Added "wire" declaration in Verilog output.

2.3: Fixed option to select or disable output registers.

2.2: Additional device support.

2.1: Miscellaneous GUI updates.

2.0: Updated for compatibility with IPexpress.

1.0: Original released version.
