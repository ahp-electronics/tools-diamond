LFSR

Description:

Implements a Linear Feedback Shift Register (LFSR).

Devices Supported:

ECP5U, ECP5UM, LatticeEC, LatticeECP, LatticeECP2, LatticeECP2M, LatticeECP3, LatticeSC, LatticeSCM, LatticeXP, LatticeXP2, LIFMD, LIFMDF, MachXO, MachXO2, MachXO3D, MachXO3L, Platform Manager

Version:

3.8

Revision History:

3.8: Added LIFMDF support.

3.7: Added LIFMD support. Added MachXO3D support.

3.6: Added ECP5 support.

3.5: Added MachXO3L support.

3.4: Added testbench template.

3.3: Added "wire" declaration in Verilog output.

3.2: Added testbench template.

3.1: Additional device support.

3.0: User has ability to configure the LFSR by selecting the implementation (Galois/Fibonacci), gate type (XOR/XNOR), and specify the feedback polynomial.

2.0: Updated for compatibility with IPexpress.

1.0: Original released version.
