Complex Multiplier

Description:

Complex Multiplier takes two inputs and multiplies them. For example, y1 = a1 + jb1 and y2 = a2 + jb2. The product is a1a2 - b1b2 + j(a1b2 + a2b1).

Look-up table (LUT) implementation is available. For DSP devices, DSP implementation is also available.

Devices Supported:

ECP5U, ECP5UM, LatticeEC, LatticeECP, LatticeECP2, LatticeECP2M, LatticeECP3, LatticeSC, LatticeSCM, LatticeXP, LatticeXP2, LIFMD, LIFMD, MachXO, MachXO2, MachXO3D, MachXO3L, Platform Manager, Platform Manager 2
