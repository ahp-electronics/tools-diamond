SDR

Description:

Allows you to generate a Single Data Rate Registers of the specified width and mode. Although an SDR generated for a specific width can be used for other widths, it usually requires extra care in specifying the connection while instantiating.

Devices Supported:

ECP5U, ECP5UM, LatticeECP3, LatticeSC, LatticeSCM, LIFMD, LIFMD, MachXO2, MachXO3D, MachXO3L, Platform Manager 2

Version:

6.1
