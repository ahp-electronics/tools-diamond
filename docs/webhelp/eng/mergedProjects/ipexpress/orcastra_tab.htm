ORCAstra

Description:

Implements an appropriate interface for the ORCAstra System Bus Control Center. For LatticeSC/M, the interface is for the User Master Interface of the System Bus. For LatticeECP2M, LatticeECP3, and LatticeXP2, the interface is for the SERDES Client Interface (SCI).

Devices Supported:

LatticeECP2M, LatticeECP3, LatticeSC, LatticeSCM, LatticeXP2

Version:

1.0

Revision History:

1.0: Original released version.
