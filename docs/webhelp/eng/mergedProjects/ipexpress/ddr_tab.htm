DDR

Description:

Allows you to generate a DDR of the specified width and mode. Although a DDR generated for a specific width can be used for other widths, it usually requires extra care in specifying the connection while instantiating.

Devices Supported:

LatticeSC, LatticeSCM

Version:

4.2

Revision History:

4.2: Changed VHDL "dont_touch" attribute to boolean instead of string.

4.1: Added "wire" declaration in verilog output.

4.0: Added RSTAIL pin for AIL mode; added output clock mode.

3.2: Added single clock option for x1 gear inputs and bidirectional.

3.1: The CLKMODE attribute for output registers is removed. (It is available only for input registers).

3.0: Added CLKMODE to allow the user to specify IO clock routing.

2.0: Updated for compatibility with IPexpress.

1.0: Original released version.
