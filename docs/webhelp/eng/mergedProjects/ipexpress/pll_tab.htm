PLL

Description:

Implements a Phase Locked Loop (PLL).

Devices Supported:

ECP5U, ECP5UM, LatticeEC, LatticeECP, LatticeECP2, LatticeECP2M, LatticeECP3, LatticeSC, LatticeSCM, LatticeXP, LatticeXP2, LIFMD, LIFMDF, MachXO, MachXO2, MachXO3D, MachXO3L, Platform Manager 2

Version:

5.8

Revision History:

5.8: Improved divider calculation speed. Added LIFMD support. Added MachXO3D support.

5.7: Bug fixes for ECP5 frequency calculation.

5.6: Added IO_TYPE support to the reference clock CLKI. Added ECP5 support.

5.5: Added MachXO3L support.

5.4: Added bandwidth calculation.

5.3: LatticeECP4 update.

5.2: Added FEEDBK_PATH for LatticeECP3.

5.1: Modified frequency range per latest LatticeECP3 data sheet. Added support for CLKOS feedback for LatticeECP2/M. Changed VHDL "dont_touch" attribute to boolean instead of string.

5.0: Modified frequency range per latest LatticeECP3 data sheet. It now allows configuration for SMPTE settings. Fixed PLL phase control setting (when CLKOS is not used) for LatticeXP2.

4.2: Fixed CLKOS feedback for LatticeECP3.

4.1: For LatticeECP2, the dialog box was changed to suggest a CLKOP divider for an optimal fVCO. For LatticeECP3, added support for CLKOS feedback. For LatticeSC, fixed issue with low frequency input. For LatticeXP2, fixed default CLKOP_TRIM_DELAY.

4.0: Added Bypass mode for LatticeSC/M. Fixed issue with LatticeSC/M low frequency inputs (2-15 MHz). For LatticeECP2, added user control for SPLL and GPLL. Fixed defaults for LatticeXP2 (duty trim). Added "wire" declaration in Verilog output.

3.7: Modified engine to generate error (rather than illegal netlist) for LatticeSC/M PLLs in low frequency with high phase (315) for CLKOS.

3.6: Added syn_noprune for I/O Assistant flow (LatticeECP2). Updated PLL range (ECP2*). Fixed issues with LatticeECP2S and LatticeECP2MS.

3.5: Added option for high bandwidth.

3.4: For LatticeSC/M, the frequency for CLKI and user clock can be as low as 2 MHz (previous lower limit was 15 MHz).

3.3: For LatticeECP2, when using static delay, netlist now instantiates an EHXPLLD element.

3.2: Additional device support.

3.1: LatticeSC/M: Changed divider calculation to set the internal VCO to operate at a lower value. LatticeECP/EC, LatticeXP: Minor bug fixes for divider mode. LatticeECP2: Changed algorithm to generate EPLLD over EHXPLLD.

3.0: LatticeSC: Changed divider calculation to set the internal VCO to operate at a higher value. LatticeECP2: minor fixes for external capacitor mode.

2.0: Updated for compatibility with IPexpress.

1.0: Original released version.
