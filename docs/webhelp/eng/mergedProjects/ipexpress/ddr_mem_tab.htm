DDR_MEM

Description:

Generates a module that can be used to interface a DDR Memory. It generates the bidirectional port and the associated clocking scheme.

Devices Supported:

ECP5U, ECP5UM, LatticeEC, LatticeECP, LatticeECP2, LatticeECP2M, LatticeECP3, LatticeXP, LatticeXP2, Platform Manager 2
