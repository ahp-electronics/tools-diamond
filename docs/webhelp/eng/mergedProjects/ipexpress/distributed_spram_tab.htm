Distributed_SPRAM

Description:

Implements a Single Port RAM using LUT logic.

Devices Supported:

ECP5U, ECP5UM, LatticeEC, LatticeECP, LatticeECP2, LatticeECP2M, LatticeECP3, LatticeSC, LatticeSCM, LatticeXP, LatticeXP2, LIFMD, LIFMDF, MachXO, MachXO2, MachXO3D, MachXO3L, Platform Manager, Platform Manager 2

Version:

4.0

Revision History:

4.0: Added LIFMDF support.

3.9: Added LIFMD support. Added MachXO3D support.

3.8: Added ECP5 support.

3.7: Added MachXO3L support.

3.6: Added support for comments and empty lines in memory initialization files.

3.5: For LatticeSC: Set GSR to ENABLED (default setting for primitive).

3.4: Added attributes to allow post-par memory reinitialization for LatticeECP3, LatticeSC, and LatticeXP2.

3.3: Added "wire" declaration in Verilog output.

3.2: Added support for initialization for LatticeXP2.

3.1: Additional device support.

3.0: Removed memory initialization.

2.1: Miscellaneous GUI updates.

2.0: Updated for compatibility with IPexpress.

1.0: Original released version.
