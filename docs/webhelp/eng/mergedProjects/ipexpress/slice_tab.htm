SLICE

Description:

Generic module used to generate custom configurations. Provides a variety of multiply/accumulate organizations and is implemented as embedded sysDSP Blocks.

Devices Supported:

ECP5U, ECP5UM, LatticeECP3

Version:

1.6

Revision History:

1.6: Bug fixes for preadders.

1.5: Added ECP5 support.

1.4: Bug fixes and minor updates.

1.3: Fixed behavior of MCPAT_SOURCE/MASKPAT_SOURCE selection. These are not allowed when the C Input is set to C_ALU or A_ALU, and both cannot be DYNAMIC at the same time.

1.2: Fixed register setting on Advanced tab based on selections of first tab.

1.1: Removed GSR option.

1.0: Original released version.
