Counter

Description:

Implements an up, up-down or down counter.

Devices Supported:

ECP5U, ECP5UM, LatticeEC, LatticeECP, LatticeECP2, LatticeECP2M, LatticeECP3, LatticeSC, LatticeSCM, LatticeXP, LatticeXP2, LIFMD, LIFMDF, MachXO, MachXO2, MachXO3D, MachXO3L, Platform Manager, Platform Manager 2

Version:

4.7

Revision History:

4.7: Added LIFMDF support.

4.6: Added LIFMD support. Added MachXO3D support.

4.5: Added ECP5 support.

4.4: Added MachXO3L support.

4.3: Fixed dummy cell connection for FCIN.

4.2: Added 64-bit support.

4.1: Added "wire" declaration in verilog output.

4.0: Added enable load input option; added upper and lower counter values; added testbench template.

3.2: Fixed problem with module not generating if installation path was too long.

3.1: Additional device support.

3.0: Added option to generate look-ahead counters. For counters with width greater than 32, the speed option is available.

2.0: Updated for compatibility with IPexpress.

1.0: Original released version.
