RAM_Based_Shift_Register

Description:

Implements a shift register using RAM and implemented in LUT logic.

Devices Supported:

ECP5U, ECP5UM, LatticeEC, LatticeECP, LatticeECP2, LatticeECP2M, LatticeECP3, LatticeSC, LatticeSCM, LatticeXP, LatticeXP2, LIFMD, LIFMDF, MachXO, MachXO2, MachXO3D, MachXO3L, Platform Manager, Platform Manager 2

Version:

5.3

Revision History:

5.3: Added LIFMDF support.

5.2: Added LIFMD support. Added MachXO3D support.

5.1: Added ECP5 support.

5.0: Added MachXO3L support. Added EBR implementation.

4.2: Added support for comments and empty lines in memory initialization files.

4.1: For LatticeSC: Set GSR to ENABLED (default setting for primitive) for LUT-based implementation.

4.0: Removed support for EBR-based shift-registers.

3.4: Added "wire" declaration in Verilog output.

3.3: Updated device support.

3.2: Additional device support.

3.1: Minimum number of shifts changed to 2 instead of 1.

3.0: Added lossless mode. Added EBR-based shift register.

2.0: Updated for compatibility with IPexpress.

1.0: Original released version.
