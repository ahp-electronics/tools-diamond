MULT

Description:

Multiplier implemented using the DSP.

Devices Supported:

ECP5U, ECP5UM, LatticeECP, LatticeECP2, LatticeECP2M, LatticeECP2S, LatticeECP2MS, LatticeECP3, LatticeXP2

Version:

4.2

Revision History:

4.2: Added ECP5 support.

4.1: Fixed dummy cell connection for FCIN.

4.0: Added fully pipelined mode. For LatticeECP3: Added GUI check to require use of input register when shift input is used.

3.2: Removed GSR option.

3.1: Added "wire" declaration in Verilog output.

3.0: Changed the size of shift-in and shift-out ports. These ports are now always 18-bits wide (regardless of input size). This change was required to match hardware architecture.

2.3: Added testbench template.

2.2: Additional device support.

2.1: Miscellaneous GUI updates.

2.0: Updated for compatibility with IPexpress.

1.0: Original released version.
