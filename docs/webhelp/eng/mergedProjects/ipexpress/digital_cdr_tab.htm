Low-Speed CDR

Description:

The CDR (Clock and Data Recovery) logic takes a clock at approximately the same frequency as the incoming data, then produces this same data re-timed with the appropriate clock phase. Features include:

  • Capable of retiming data from 100 MHz to 500 MHz.
  • Can accommodate jitter components up to 0.25 UI.
  • No additional logic required (FIFOs built-in).
  • Can handle up to 18 channels of data.
