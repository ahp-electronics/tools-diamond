EHXPLLL is the GPLL primitive for ECP5. The following are descriptions of EHXPLLL port functions.

Port
I/O
Function
CLKI
I
Input Clock to PLL.
CLKI2
I
Muxed Input Clock to PLL.
SEL
I
Select Clock
CLKFB
I
Feedback Clock.
PHASESEL[1:0]
I
Select the output affected by Dynamic Phase adjustment.
PHASEDIR
I
Dynamic Phase adjustment direction.
PHASESTEP
I
Dynamic Phase adjustment step.
PHASELOADREG
I
Load dynamic phase adjustment values into PLL.
CLKOP
O
PLL main output clock.
CLKOS
O
PLL output clock.
CLKOS2
O
PLL output clock.
CLKOS3
O
PLL output clock.
LOCK
O
PLL LOCK to CLKI, Asynchronous signal. Active high indicates PLL lock.
STDBY
I
Standby signal to power down the PLL.
RST
I
Resets the whole PLL.
ENCLKOP
I
Enable PLL output CLKOP
ENCLKOS
I
Enable PLL output CLKOS
ENCLKOS2
I
Enable PLL output CLKOS2
ENCLKOS3
I
Enable PLL output CLKOS3
