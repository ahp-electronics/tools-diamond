ODDRX8A

Architectures Supported:

LIFMD
LIFMDF

INPUT: D0, D1, D2, D3, D4, D5, D6, D7, D8, D9, D10, D11, D12, D13, D14, D15 ECLK , SCLK, RST

OUTPUT: Q

Description

This primitive is used for 16:1 LVDS ODDR implementation

See the below table for the port description.
