EHXPLLM

GPLL

Architectures Supported:

LIFMD
LIFMDF

INPUTS: CLKI, CLKFB, PHASESEL1, PHASESEL0, PHASEDIR, PHASESTEP, PHASELOADREG, STDBY, PLLWAKESYNC, RST, ENCLKOP, ENCLKOS, ENCLKOS2, ENCLKOS3, USRSTDBY

OUTPUTS: CLKOP, CLKOS, CLKOS2, CLKOS3, LOCK, INTLOCK, REFCLK, CLKINTFB

EHXPLLM is the GPLL primitive for LIFMD/F. The following are descriptions of EHXPLLM port functions

Port
IO
Description

I
Input Clock to PLL.

I

SEL

I
Feedback Clock.

I
Select the output affected by Dynamic Phase adjustment.

I
Dynamic Phase adjustment direction.
PHASESTEP
I
Dynamic Phase adjustment step.
PHASELOADREG
I
Load dynamic phase adjustment values into PLL.

CLKOS
O
PLL output clock.

O
PLL output clock.
CLKOS3
O
PLL output clock.

PLL LOCK to CLKI, Asynchronous signal. Active high indicates PLL lock.
