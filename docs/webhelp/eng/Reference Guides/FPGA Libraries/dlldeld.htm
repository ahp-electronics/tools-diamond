DLLDELD

Slave Delay

Architectures Supported:

ECP5
LIFMD
LIFMDF

INPUTS: A, DDRDEL, LOADN, MOVE, DIRECTION

OUTPUT: Z, CFLAG

Description

DLLDELD is the delay element that receives code from DDRDLL for generic DDR. The DLLDELD port description is shown in the below table.

Port Name
I/O
Description
A
I
Clock Input
DDRDEL
I
Delay inputs from DDRDLL
LOADN
I
Used to reset back to 90 degrees delay.
MOVE
I
Pulse is required to change delay settings. The value on Direction will be sampled at the “falling edge” of MOVE.
DIRECTION
I
Indicates delay direction. “1” to decrease delay & “0” to increase delay.
CFLAG
O
Indicates the delay counter has reached max value when moving up or min value when moving down.
Z
O
Delayed Clock output
