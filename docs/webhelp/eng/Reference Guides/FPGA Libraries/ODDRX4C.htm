ODDRX4C

Architectures Supported:

LIFMD
LIFMDF

INPUT: D0, D1, D2, D3, D4, D5, D6, D7, ECLK , SCLK, RST

OUTPUT: Q

Description

This primitive is used for 8:1 LVDS ODDR implementation.

See the below table for the port description.

Signal
I/O
Description
D0...D7
I
Data input to the ODDR
ECLK
I
ECLK input (4x speed of SCLK)
SCLK
I
SCLK input
RST
I
Reset Input
Q
O
DDR data output on both edges of ECLK
