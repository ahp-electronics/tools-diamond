ODDRX71A

7:1 LVDS Output

Architectures Supported:

LFMNX
MachXO2
MachXO3D
MachXO3L
Platform Manager 2

INPUTS: ECLK, SCLK, D0, D1, D2, D3, D4, D5, D6, RST

OUTPUT: Q

ATTRIBUTES:

GSR: "ENABLED" (default), "DISABLED"

Description

ODDRX71A is the 7:1 LVDS output that supports 7:1 gearing. It is used for top bank only. See the below table for the port description.

Signal
I/O
Description
ECLK
I
Edge clock
SCLK
I
Clock connected to the system clock
D0, D1, D2, D3, D4, D5, D6
I
Data available for 7:1 muxing
RST
I
RESET for this block
Q
O
7:1 LVDS signal output
