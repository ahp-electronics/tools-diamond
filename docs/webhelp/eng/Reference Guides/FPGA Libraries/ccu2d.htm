CCU2D

Carry Chain

Architectures Supported:

LFMNX
MachXO2
MachXO3D
MachXO3L
Platform Manager 2

INPUTS: CIN, A1, B1, C1, D1, A0, B0, C0, D0

OUTPUTS: S1, S0, COUT

ATTRIBUTES:

INIT0: hexadecimal value (default: 16’h0000)

INIT1: hexadecimal value (default: 16’h0000)

INJECT1_0: "YES" (default), "NO"

INJECT1_1: "YES" (default), "NO"
