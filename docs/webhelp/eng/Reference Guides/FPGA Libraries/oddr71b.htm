ODDR71B

7:1 LVDS ODDR implementation

Architectures Supported:

ECP5
LIFMD
LIFMDF

INPUTS: D0, D1, D2, D3, D4, D5, D6, ECLK, SCLK, RST

OUTPUT: Q

Description

This primitive is used for 7:1 LVDS ODDR implementation.

I/O
