FSUB2

2 Bit Fast Subtractor (two's complement)

Architectures Supported:

LatticeECP/EC
LatticeSC/M
LatticeXP
MachXO
Platform Manager

INPUTS: A1, A0, B1, B0, BI

OUTPUTS: BOUT1, BOUT0, S1, S0

Description

FSUB2 is a 2-bit twoâ€™s complement subtractor. It has a borrow-in input (BI) and two 2-bit input (A0, A1 and B0, B1). The FSUB2 produces a 2-bit difference output (S0, S1) along with a 2-bit borrow-out output (BOUT1, BOUT).
