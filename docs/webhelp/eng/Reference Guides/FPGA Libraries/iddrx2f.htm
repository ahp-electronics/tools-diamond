IDDRX2F

Generic input DDR Primitive

Architectures Supported:

ECP5
LIFMD
LIFMDF

INPUTS: D, SCLK, RST, ECLK, ALIGNWD

OUTPUTS: Q0, Q1, Q2, Q3

Description

This primitive is used for Generic X1 IDDR implementation. The following table gives the port description of the IDDRX2F primitive.

Port Name
I/O
Definition
D
I
DDR Data input.
SCLK
I
Primary clock input (divide by 2 of ECLK).
RST
I
Reset to DDR registers.
ECLK
I
Fast Edge clock.
ALIGNWD
I
This signal is used for Word alignment. It will shift word by one bit.
Q0, Q2
O
Data at positive edge of input ECLK.
Q1, Q3
O
Data at negative edge of input ECLK.
