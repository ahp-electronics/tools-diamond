IDDRX1F

Generic input DDR Primitive

Architectures Supported:

ECP5
LIFMD
LIFMDF

INPUTS: D, SCLK, RST

OUTPUTS: Q0, Q1

Description
This primitive is used for Generic X1 IDDR implementation. The following table gives the port description of the IDDRX1F primitive.
Port Name
I/O
Definition
D
I
DDR Data input.
SCLK
I
Primary clock input.
RST
I
Reset to DDR registers.
Q0
O
Data at positive edge of clock.
Q1
O
Data at negative edge of clock.
