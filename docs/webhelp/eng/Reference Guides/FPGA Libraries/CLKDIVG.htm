CLKDIVG

Clock Divider

Architectures Supported:

LIFMD
LIFMDF

INPUTS: CLKI, RST, ALIGNWD

OUTPUTS: CDIVX

ATTRIBUTES:

GSR: "DISABLED" (default), "ENABLED"

DIV: 2.0 (default)
