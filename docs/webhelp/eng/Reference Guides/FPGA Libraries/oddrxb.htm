ODDRXB

Output DDR

Architectures Supported:

LatticeECP/EC
LatticeXP

INPUTS: DA, DB, CLK, LSR

OUTPUT: Q

ATTRIBUTES:

REGSET: "RESET" (default), "SET"

Description

Output DDR data with half cycle clock domain transfer.
