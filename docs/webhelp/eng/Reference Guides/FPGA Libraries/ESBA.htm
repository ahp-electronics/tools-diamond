I/O
Description

Write Data Input. Split transactions are not supported requiring the master to stay active on the bus until a valid response is received.
WBDATO[31:0]
O
Read Data Output
WBSTBI
I
Strobe input indicating the WB slave is the target for the current transaction on the WB bus.
WBACKO
O
Transfer Acknowledge asserted by the WB slave to the master, indicating the requested transfer has been completed. This signal is qualified by WBSTBI.
WBCLKI
I
Positive edge clock used by all WB interface logic blocks.

Indicates FIFO is full.
ASFCLKI
I
Clock for ASF.

I
Resets the WB logic.

Resets the ASF logic.
