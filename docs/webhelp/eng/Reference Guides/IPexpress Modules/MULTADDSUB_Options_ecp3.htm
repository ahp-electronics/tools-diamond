MULTADDSUB Options (ECP5, LatticeECP3)

MULTADDSUB Dialog Box for ECP5 and LatticeECP3 
Feature
Description
Size of the DSPMULTADDSUB block
Enable Cascade Input – Select if this module will also get input from another MULTADDSUB module. Enabling cascade input reduces the number of bits available for the multiplier input buses (below).
The output width is always 54 bits. This is needed to feed into the 54-bit CIN of the next slice. For the final slice of the chain, some lower bits may have to be ignored. Due to the alignment of the inputs to the primitives, the lower (36 - (A0/A1 width + B0/B1 width)) must be ignored. If both the input widths are less than or equal to 9, then the lower (18 - (A0/A1 width + B0/B1 width)) must be ignored. For example, if Input A0/A1 width = 12 and Input B0/B1 width = 16, the lower 36 - (12+16) = 8 bits should be ignored.
Input A0/A1 width – Specify the size of the first term in the multiplication (for both multipliers).
Input B0/B1 width – Specify the size of the second term in the multiplication (for both multipliers).
Sum output width – Size of the output. This is one more than the product size (that is, it is equal to A0/A1 size + B0/B1 size + 1).
Select Options
Select Shift Out A - Enables the shift-out port A of the DSP block.
Cascade Match Register – Enables the Cascade Match Register of the A multiplier.
Select Shift Out B - Enables the shift-out port B of the DSP block.
Add/Sub Operation - Select if the adder is adding, subtracting or dynamic mode. In dynamic mode high = add, low = subtract.
Data Options
Input A0, A1, B0, B1 Source – Specify if the source of the input is Parallel for generic routing, Shift for the shift outputs of an adjacent DSP block, or Dynamic for dynamic selection of the source of the input.
Data Type A, B – Specify the data format of the inputs as signed, unsigned, or dynamic selection of the sign.
Register Options
Reset Mode – Specify whether the reset is synchronous or asynchronous.
Clock – Selects the clock for the registers of the DSP block.
CE – Selects the clock enable pin to the registers of the DSP block.
RST – Selects the reset pin to the registers of the DSP block.
Bus Ordering Style
Specify if the inputs and output are big endian (MSB:LSB) or little endian (LSB:MSB), or none.
Import IPX to Diamond project
Adds the customized module to the currently open project. Not available if running IPexpress as a stand-alone tool. Not available in Clarity Designer.
Generate
Builds the module as specified.
Close
Closes the dialog box without generating the module.
Help
Opens the online Help.
