Adder_Subtractor

A two-input adder/subtractor that performs signed or unsigned addition/subtraction of the data from input ports DataA and DataB based on the specification from the input port Add_Sub. If the optional port Cin is used, the addition/subtraction result DataA + DataB + Cin / DataA - DataB + Cin - 1 are returned to the output port Result. An optional Overflow port is provided to detect overflow conditions when doing operations on signed data. or a description of some of the possible ports, see this table.

Adder_Subtractor Ports
Name
Description
Add_Sub
Add_Sub=1, then Result=DataA+DataB+Cin.
Add_Sub=0, then Result=DataA-DataB+Cin-1.
Cin (optional)
Carry-in to the low-order bit.
DataA[size-1..0]
Augend (Add_Sub=1) / Minuend (Add_Sub=0)
DataB[size-1..0]
Addend (Add_Sub=1) / Subtrahend (Add_Sub=0)
Result[size-1..0]
Result (Add_Sub=1) = DataA + DataB + Cin
Result (Add_Sub=0) = DataA - DataB + Cin - 1
Cout (optional)
Carry-out of the MSB to detect overflow in unsigned representation. When Cout is prepended to the Result, the result is a vector that always has sufficient precision to represent the result of the operation. {Cout, Result} = DataA + DataB + Cin (Add_Sub=1)
{Cout, Result} = DataA - DataB + Cin - 1 (Add_Sub=0)
Overflow (optional)
Result exceeds available precision when used with signed representation. XOR of the carry into the MSB and Cout.
