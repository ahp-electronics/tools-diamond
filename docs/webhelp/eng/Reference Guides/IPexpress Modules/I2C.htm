I2C

A System Bus Interface (SBI) or FIFO Interface connects the Hard IP to the fabric. The system bus is connected to two I2C Hard IPs. Select an option in the first tab to enable the other two tabs.

I2C Dialog Box—I2C Settings Tab
Feature
Description
General
General Call Enable – Enables I2C General call support. Sets a register based on the active mode (FIFO or System bus).
Include IO Buffers – Include buffers for the I2C pins. The pins can be placed in any IO location, and will default to the IO LVCMOS standard. Includes the I2C pins I2Cx_SCL and I2Cx_SDA.
SDA Input Delay (ns) – Add a 50 ns input delay.
SDA Output Delay (ns) – Add a 50, 75, 125, 150, 200, 300, or 350 ns output delay.
Master Clock Rate
Desired(KHz) – Enter a desired clock rate in KHz to automatically determine a divider value and generate a clock close to this value based on the input clock. The frequency of this input System Bus clock is specified in the Hard IP Enables tab. The divider is rounded to the nearest integer after dividing the input System Bus clock by the value entered in this field.
Actual(KHz) – Since the input System Bus clock cannot always be divided into the user’s exact desired master clock, this field displays the actual value. This read-only value is calculated as follows:
Actual clock = rounded(System Bus clock /
rounded(System Bus clock / Master clock)
)
Divider values exactly halfway between two integers are rounded up (for example, a divider value of 108.5 is rounded to 109).
FIFO Mode
Enable FIFO Mode – Allows you change all bits of the I2C address.
CLK Stretch Enable – Enables data setup time.
RxFIFO Almost Full – Shows the actual almost full value (not the offset), determined by I2CFIFOTHRESHOLD[9:5] = 32 - I2C_FIFO_RxAlmostF.
TxFIFO Almost Empty – Shows the actual almost empty value.
I2C Addressing
Addressing mode – Select 7-bit or 10-bit addressing.
Address – Define the Hard I2C address. FIFO mode allows all bits to be programmed, but register mode fixes the lower two bits.
PMU Wake Options
Address Match – Wake up the PMU on Address Match.
RxFIFO Almost Full – Only available for I2C0, as that is the only I2C block connected to the PMU. Enabling this option also enables the Glitchless Clock Switch.
