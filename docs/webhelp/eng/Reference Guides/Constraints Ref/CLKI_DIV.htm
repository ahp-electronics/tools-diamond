CLKI_DIV

Convention

CLKI_DIV= value

Description

Input clock divider setting attached to a PLL element such as EHXPLLB. For LatticeSC/M and LatticeECP2/3 devices, can also be attached to a DLL element such as CIDLLA.

Device Support

LatticeSC/M, LatticeECP2/3

Examples

For generic examples of how to use FPGA attributes in your HDL, see Adding FPGA Attributes to HDL.
