INITVAL

Convention

INITVAL= value

Description

Specifies the initialization value for memory elements such as SPR16X2 and DPR16X2. These elements carry prescribed initialization values. For example, DPR16X2 has an initialization value of 0x0000000000000000.

Refer to the FPGA Libraries Reference Guide.

for more details on memory elements and the INITVAL attribute.
