CLKFB_FDEL

Convention

CLKFB_FDEL= 0, 100, 200,...700

Description

CLKFB Fine Delay Setting. Attached to a PLL element (e.g., EHXPLLA). Attribute can take the integer value set of 0, 100, 200,...700.

Device Support

LatticeSC/M
