ECLKDEL

Convention

ECLKDEL= DEL0|1|2|3

Description

Used for the EC Clock, an ECLKDEL attribute may be placed on input DDR registers for specifying cycle stealing. The attribute belongs on DDR instances.

Related Attributes:

CLKIDEL
CLKODEL
CLKDELAY
Device Support

LatticeSC/M, LatticeXP/2, LatticeECP/EC

VHDL Syntax

ATTRIBUTE ECLKDEL : string; ATTRIBUTE ECLKDEL OF [pin_type]: SIGNAL IS "[signal_name]";

VHDL Example Code
attribute eclkdel : string;
attribute eclkdel of input_vector : signal is "DEL0";
Verilog Syntax – Precision

//pragma attribute [pin_type] ECLKDEL [signal_name]

Verilog Example Code – Precision
//pragma attribute load eclkdel del1
Verilog Syntax – Synplify

PinType [pin_type] /* synthesis ECLKDEL="[signal_name]"*/;

Verilog Example Code – Synplify
//input load /* synthesis ECLKDEL="DEL2" */;
ECLKDEL Written Out to Preference (.prf) File

Here is an example of how ECLKDEL attributes might appear after they are written out to the preference file as the CLKDELAY logical preference in the post-map phase:

CLKDELAY CELL "eclkdel0" DEL0 ;
