FBEXTDELAY PCM

Specifies the external feedback additional delay on a PCM in PLL mode.

Device Support

All

Syntax

FBEXTDELAY PCM <pcm name> (FROM PORT <name> TO PORT <name>) <time>;

Example
FBEXTDELAY PCM "CLKGEN/PLLDInst_0" FROM PORT "clkout" TO PORT "clkfb" 1 ns;
