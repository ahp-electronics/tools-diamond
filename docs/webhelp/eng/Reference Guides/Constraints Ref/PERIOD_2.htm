PERIOD

Convention

PERIOD = xxx

Description

Clock net attribute used to identify a clock period for all pins clocked by the specified net. If the net is optimized away, no preference is generated. Translated by the mapper into the PERIOD preference. The xxx value is expressed in ps (the resultant preference is expressed in ns).

Device Support

All
