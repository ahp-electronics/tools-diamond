PHASEADJ

Convention

PHASEADJ= value

Description

(LatticeSC/M) VCO Output Phase. Attached to a PLL element (e.g., EHXPLLA), the PHASEADJ attribute adjusts phase relative to CLKOP. PHASEADJ takes the value parameter set of 0, 45, 90, 135, 190, 225, 270, or 315.

(LatticeXP/2, LatticeECP2/M, LatticeECP/EC) Coarse Phase Shift Selection. Attached to a PLL element (e.g., EHXPLLB). Attribute takes the value parameter set of 0, 45, 90, 135, 190, 225, 270, or 315.

Default value is 0.

Device Support

LatticeSC/M, LatticeXP/2, LatticeECP2/M, LatticeECP/EC
