DEFINE BUS

Specifies a grouping of nets to be used in the following preferences: BLOCK, MAXDELAY, MAXSKEW, PRIORITIZE.

Device Support

All

Syntax

DEFINE BUS <bus_name> ( NET <net_name> )+;

Examples

The following command defines a bus named busA composed of eight nets named net pen (0-7):

DEFINE BUS "busA" NET "pen0" NET "pen1" NET "pen2" NET "pen3" NET "pen4" NET "pen5" NET "pen6" NET "pen7";
