REGION

Convention

REGION = <identifier>

Description

REGION indicates the region to which a given HGROUP or UGROUP belongs. This attribute must appear on a block that has a HGROUP or UGROUP attribute. This has replaced the old PREGION attribute which is also still valid for backwards compatibility.  

Device Support

All

Examples

For generic examples of how to use FPGA attributes in your HDL, see Adding FPGA Attributes to HDL.
