SPREAD_DRIFT

Convention

SPREAD_DRIFT= 1, 2, 3

Description

Spread Spectrum BW. Attached to the PLLA element, the SPREAD_DRIFT attribute takes integer parameters of 1, 2, or 3.

Device Support

LatticeSC/M
