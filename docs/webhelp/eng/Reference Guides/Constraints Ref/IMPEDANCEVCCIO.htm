IMPEDANCEVCCIO

Convention

IMPEDANCEVCCIO= OFF (default), 16.7, 20, 25, 33, 50, 100 (Impedance up)

Description

Attached to output and bidirectional buffers, this attribute is OFF by default. This attribute sets the on chip series terminations. This programmable option can be set for each of the I/O individually. This attribute sets a pullup series impedance.

The 25, 20 and 16.7ohm terminations are only available on the primary pads of the PIO, the rest are available for both primary and alternate pads. The impedance for VCCIO can be set to the values of 16.7, 20, 25, 33, 50 and 100.

Device Support
