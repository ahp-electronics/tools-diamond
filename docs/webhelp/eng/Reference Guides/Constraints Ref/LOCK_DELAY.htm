LOCK_DELAY

Convention

LOCK_DELAY= <integer>

Description

This is a PLL lock time attribute used for simulation. Valid integer values are 0 to 100. The default value is 100 ns. This attribute is optional. If you wish to enter other than the default value of 100 ns, it can be done by adding this attribute in the DEFPARAM section of the code generated by IPexpress. You can also set this attribute in Spreadsheet View.

Device Support

LatticeEC, LatticeECP/2/3, LatticeSC/M, LatticeXP/2
