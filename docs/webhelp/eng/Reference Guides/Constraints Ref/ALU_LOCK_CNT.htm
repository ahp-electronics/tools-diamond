ALU_LOCK_CNT

Convention

ALU_LOCK_CNT= 3, 4, 5,...15

Description

Lock Count Cycles. Attached to a DLL element. Takes the integer set of 3 to 15.

Device Support

LatticeECP3, LatticeSC/M
