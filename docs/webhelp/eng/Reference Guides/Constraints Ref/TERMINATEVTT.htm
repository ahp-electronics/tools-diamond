TERMINATEVTT

Convention

TERMINATEVTT= OFF (default), 40, 50, 60, 75, 120, 150, 210

Description

This attribute sets the on-chip input parallel termination to VTT. This programmable option can be set for each of the I/O individually. When VCCIO is 3.3V, parallel input termination to VTT is still supported in the bank, as long as the PAD itself is not at 3.3 volts. See the LatticeSC PURESPEED I/O Usage Guide for  more information on this and related attributes.

Device Support

LatticeSC/M, LatticeECP3
