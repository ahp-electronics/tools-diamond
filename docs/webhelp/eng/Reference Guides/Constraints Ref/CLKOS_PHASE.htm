CLKOS_PHASE

Convention

CLKOS_PHASE= 0, 90, 180, 270, 360

Description

CLKOS Phase Select. Attached to a PLL element (e.g., EHXPLLA). Attribute can take the integer value set of 0, 90, 180, 270, 360 degrees. CLKOS_PHASE and CLKOS_VCODEL.  These two types of delays are additive to create the final delay. CLKOS_PHASE works in terms of degrees of the VCO period. CLKOS_VCODEL works in terms of full VCO periods.

Device Support

LatticeSC/M, LatticeECP2/M, LatticeECP3
