NOM_FREQ

Convention

NOM_FREQ= value

Description

Nominal Frequency selection (MHz) for oscillator elements OSC and OSCD. Values are 2.5 (default), 4.3, 5.4, 6.9, 8.1, 9.2, 10.0, 13.0, 15.0, 20.0, 26.0, 30.0, 34.0, 41.0, 45.0, 55.0, 60.0, and 130.0.

Device Support

LatticeECP2/M, LatticeECP3
