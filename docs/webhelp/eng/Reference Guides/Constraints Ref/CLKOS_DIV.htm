CLKOS_DIV

Convention

CLKOS_DIV= 1, 2, 3,...63, 64

Description

CLKOS Divider Value Select. Attached to a PLL element (e.g., EHXPLLA). Attribute can take the integer values of  1-64.

Device Support

LatticeSC/M, LatticeECP2/M, LatticeECP3, ECP5
