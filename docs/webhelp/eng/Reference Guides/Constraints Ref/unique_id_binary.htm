TRACEID

Specifies a code for storing device data, much like USERCODE but in binary format.

Device Support

MachXO2, MachXO3D, MachXO3L, MachXO3D, Platform Manager 2

Syntax

TRACEID<binary_string>;

where:

<binary_string> ::= binary value of up to 8 bits

Example
TRACEID "00001111";
