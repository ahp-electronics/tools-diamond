GSR

Convention

GSR= DISABLED | ENABLED

Description

Use to enable/disable the global set/reset (GSR) for all registered elements from these families. For example, this attribute is applicable to registers, PLLs, and memories such as SP8KA, DP8KA and the like.

Device Support

All

Examples

For specific usage examples for the GSR attribute, see the “Programmable GSR Verilog HDL Example for a Synchronous RTL Module” section in the How to Use the Global Set/Reset (GSR) Signal topic. For generic examples of how to use FPGA attributes in your HDL, see Adding FPGA Attributes to HDL.
