EQ_CAL

Convention

EQ_CAL=value

Description

Equalization filtering, available for single-ended inputs on both true and complementary I/Os and for differential inputs on true I/Os. Equalization is required to compensate for the difficulty of sampling alternating logic transitions with relatively slow slew rate.

Value: 0

Device Support

LatticeECP3

VHDL Syntax

ATTRIBUTE EQ_CAL : string;
ATTRIBUTE EQ_CAL OF [pin_name] : signal is “[value]”;

VHDL Example Code
attribute eq_cal: string;
attribute eq_cal of input1: signal IS "0";
Verilog Syntax – Precision

//pragma attribute pin_name EQ_CAL [value]

Verilog Example Code – Precision
//pragma attribute input1 EQ_CAL 0
Verilog Syntax – Synplify

PinType [pin_name] /* synthesis EQ_CAL ="[value]"*/;

Verilog Example Code – Synplify
