LOAD

Convention

LOAD= xx.xxx

Description

Attached to an output buffer, modifies loading on outputs from the default 50 PFs for timing analysis. Translated by the mapper into the OUTPUT LOAD preference.

The xx.xxx represents the loading on the output in picofarads. The resultant preference would be:

OUTPUT COMP <name> LOAD xx.xxx PF;
Device Support

All
