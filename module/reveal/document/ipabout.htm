FIR Filter Generator

Device Support: LatticeECP-DSP, LatticeECP2-DSP, LatticeECP2M-DSP

Overview: Finite Impulse Response (FIR) filters are a common type of filter that provides functionality to remove noise, provide spectral shaping, or perform signal detection in digital systems.  A FIR filter performs a convolution of an input data sequence with the filter's impulse response (the discrete time inverse Fourier transform of its desired frequency response), which is stored in memory.  Because of their inherently stable structure, FIR filters are used in systems that require linear phase. These characteristics have lead to FIR filters being designed into a large number of applications.  The Lattice FIR Filter Generator IP core provides designers all the requirements to generate and customize a FIR filter with a rich set of parameters, while providing system design flexibility, simplifying timing complexity, and several operating modes.

 

Features:    

  • Variable number of taps, up to 2048
  • Input and coefficients widths from 2 to 36 Bits
  • Selectable rounding: truncation, round to nearest, convergent rounding
  • Optional saturation logic for overflow handling
  • Full precision arithmetic
  • Signed or unsigned data and coefficients
  • Configurable parallelism from full to single multiplier
  • Multi-channel support (up to 256 Channels)
  • Decimation and interpolation ratios from 2 to 256
  • Configurable pipelining to increase performance
  • Specification of fractional inputs and outputs
  • Optional external memory interface for data and coefficients
  • Multi-channel operation with per-channel coefficients and parameters
  • Channel_in/channel_out ports to facilitate system timing
